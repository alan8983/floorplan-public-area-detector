## 三、1500px 訓練框架

Pipeline 原始設計基於 3000px+ 圖面，但考量到：
- 網路可取得的平面圖大多在 600-2000px
- 公共建築（醫院、市政府）導覽圖原始解析度高（7000-8000px），縮小到 1500px 仍清晰
- 建案銷售平面圖原始就只有 600-800px

**決定：以 1500x1500 為標準測試解析度。** Pipeline 參數（wall detection kernel sizes、
gap closing thresholds 等）需要按比例調整。

## 四、取得更多樣本的建議

1. **掃描紙本建照圖**：300DPI A3 掃描 → ~3500x5000px，最佳品質
2. **建築師事務所電子檔**：CAD 匯出 300DPI PNG
3. **建管處圖說複印**：台北市可臨櫃申請竣工圖複印
4. **合作夥伴**：消防設備師事務所通常有大量消防圖說
5. **公共建築**：醫院、大學、政府機關的樓層導覽圖（已驗證可取得 7000px+）
